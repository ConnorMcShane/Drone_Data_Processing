How To Use

Run in command line using "python ManualCorrectionTool.py --source videofile"

Make sure the csv file for bounding boxes is correct in the code.

To rewind video, press 'r' and make sure video is in playing mode.

To pause video, press 'p'. When paused enter Insert/Remove/Nothing into command.


When Insert is entered, select the ROI on the image, press 's' to reset the ROI and press 'c' to confirm ROI

When Remove is selected, enter the vehicle id you want to remove from the selection.

When Nothing is entered, the video restarts.


Outputs:

Updated csv file 
Change log csv file
